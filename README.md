# To-Do-List-Web-App

Simple To-Do list web application built with plain HTML, CSS and JavaScript.
Users can manage their daily tasks by adding, editing, sorting, and organizing them by category or due date.

**Live Demo:** [https://to-do-list-nattanan.netlify.app/]

Features:
- Add / edit / delete tasks
- Mark complete / uncomplete
- Filter: All / Active / Completed
- Search tasks
- Persist data in localStorage
- Set a due date for each task
- Choose a category
- Sort tasks

## Tech
- HTML5, CSS3, JavaScript (ES6), Netlify
